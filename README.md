# ❌⭕ Tic Tac Toe (Python)

Ein klassisches Tic-Tac-Toe-Spiel für 2 Spieler im Terminal.

---

# 📌 Beschreibung

Zwei Spieler spielen abwechselnd auf einem 3x3 Spielfeld.

Ziel: 3 gleiche Zeichen (X oder O) in einer Reihe, Spalte oder Diagonale.

---

# 🎮 Spielregeln

- Spieler X beginnt
- Abwechselnd X und O
- Eingabe von Zeile und Spalte (0–2)
- Nur freie Felder sind erlaubt
- Gewinner: 3 in einer Linie
- Wenn alle Felder voll → Unentschieden

---

# 🧠 Funktionen

## Spielfeld erstellen
Erstellt ein leeres 3x3 Spielfeld.

## Spielfeld anzeigen
Zeigt das aktuelle Spielfeld im Terminal an.

## Zug machen
Setzt X oder O auf ein Feld, wenn es frei ist.

## Gewinn prüfen
Prüft:
- Reihen
- Spalten
- Diagonalen

## Unentschieden prüfen
Prüft, ob das Spielfeld voll ist.

---

# ▶️ Beispiel Ablauf

Spieler X, wähle deine Zeile (0 - 2)  
Spieler X, wähle deine Spalte (0 - 2)  

X | O | X  
------  
O | X | O  
------  
X |   | O  

X hat gewonnen.
